# 快速开始

## 环境要求

- Java 17+
- Maven 3.6+（项目已包含 Maven Wrapper）
- MySQL 8.0+
- Redis 6.0+

## 环境检查

```bash
# 检查 Java
java -version

# 检查 MySQL
mysql -u root -p

# 检查 Redis
redis-cli ping
```

## 部署步骤

1. [环境配置](2.环境配置指南.md) - 配置数据库、Redis、JWT
2. [数据库部署](3.数据库部署指南.md) - 运行 `sql/setup.bat` 初始化数据库
3. [依赖安装](4.依赖安装指南.md) - 安装 Maven 依赖
4. 启动服务

```bash
# Windows
.\mvnw.cmd spring-boot:run -pl am-core

# macOS/Linux
./mvnw spring-boot:run -pl am-core
```
