# 依赖安装指南

## Maven Wrapper

项目已包含 Maven Wrapper，无需单独安装 Maven。

```bash
# 使用项目自带的 Maven Wrapper
# Windows:
.\mvnw.cmd -version

# macOS/Linux:
./mvnw -version
```

## 项目依赖安装

### 1. 安装Maven依赖
```bash
# 使用 Maven Wrapper 安装依赖
# Windows:
.\mvnw.cmd clean install

# macOS/Linux:
./mvnw clean install

# 如果网络较慢，可以使用阿里云镜像
# 在 ~/.m2/settings.xml 中配置镜像源
```

### 2. 验证依赖安装
```bash
# 检查依赖树
.\mvnw.cmd dependency:tree

# 编译项目
.\mvnw.cmd compile
```

## 依赖验证

### 验证Maven配置
```bash
# 检查Maven Wrapper是否正常工作
.\mvnw.cmd -version

# 检查项目配置是否正确
.\mvnw.cmd validate
```
