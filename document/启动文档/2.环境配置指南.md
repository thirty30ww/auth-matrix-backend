# 环境配置指南

## 配置文件

在 `am-core/src/main/resources/` 目录下创建环境配置文件：

- 开发环境：`application-dev.yml`
- 生产环境：`application-prod.yml`

## 配置模板

```yaml
spring:
  datasource:
    dynamic:
      datasource:
        user:
          username: root
          password: your_password
        system:
          username: root
          password: your_password
  data:
    redis:
      password: your_redis_password  # 无密码可删除此行

jwt:
  secret: your_jwt_secret_key  # 使用 openssl rand -base64 32 生成

aliyun:
  oss:
    endpoint: oss-cn-hangzhou.aliyuncs.com
    accessKeyId: your_access_key_id
    accessKeySecret: your_access_key_secret
    bucketName: your_bucket_name
    domain: https://your-bucket.oss-cn-hangzhou.aliyuncs.com
```

> [!NOTE]
> OSS 配置为可选项，不使用可不配置

## 环境切换

在 `application.yml` 中修改：

```yaml
spring:
  profiles:
    active: dev  # dev 或 prod
```
