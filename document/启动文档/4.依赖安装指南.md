# 依赖安装指南

## 安装依赖

项目已包含 Maven Wrapper，无需单独安装 Maven。

```bash
# Windows
.\mvnw.cmd clean install

# macOS/Linux
./mvnw clean install
```

## 配置镜像（可选）

如下载速度慢，在 `~/.m2/settings.xml` 配置阿里云镜像：

```xml
<settings>
    <mirrors>
        <mirror>
            <id>aliyunmaven</id>
            <mirrorOf>*</mirrorOf>
            <url>https://maven.aliyun.com/repository/public</url>
        </mirror>
    </mirrors>
</settings>
```

## 启动项目

```bash
# Windows
.\mvnw.cmd spring-boot:run -pl am-core

# macOS/Linux
./mvnw spring-boot:run -pl am-core
```

## 打包部署

```bash
.\mvnw.cmd clean package
java -jar am-core/target/am-core-0.0.1-SNAPSHOT-exec.jar
```
